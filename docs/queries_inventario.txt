QUERIES SQL ÚTEIS PARA O PROJETO DE INVENTÁRIO INFORMÁTICO
============================================================

NOTA:
Estas queries assumem nomes de tabelas/campos como:

Tabelas:
- inventarios
- computadores
- localizacoes
- utilizadores
- perfis
- dispositivos_descobertos
- logs_sistema
- logs_dispositivo

Campos principais esperados:

computadores:
- id
- nome
- marca
- modelo
- numero_serie
- estado
- inventario_id
- localizacao_id
- utilizador_responsavel_id

inventarios:
- id
- nome
- descricao

localizacoes:
- id
- nome
- descricao

utilizadores:
- id
- nome
- email
- perfil_id

perfis:
- id
- nome
- descricao

dispositivos_descobertos:
- id
- ip
- hostname
- mac_address
- estado
- inventario_id

Se algum nome for diferente no código real, basta adaptar a query.

---------------------------

Serve para listar todos os inventários criados.

-------------------------------

----------------------------

FROM computadores;

Lista todos os computadores registados no sistema.


4. VER TODOS OS COMPUTADORES DE UM INVENTÁRIO
---------------------------------------------

SELECT *
FROM computadores
WHERE inventario_id = 1;

Serve para responder:
Quais são os computadores que pertencem ao inventário 1?


5. VER COMPUTADORES DE UM INVENTÁRIO COM NOME DO INVENTÁRIO
-----------------------------------------------------------

SELECT
    i.id AS inventario_id,
    i.nome AS inventario_nome,
    c.id AS computador_id,
    c.nome AS computador_nome,
    c.marca,
    c.modelo,
    c.numero_serie,
    c.estado
FROM inventarios i
JOIN computadores c
    ON c.inventario_id = i.id
WHERE i.id = 1;

Serve para mostrar o inventário e os PCs associados.


6. VER COMPUTADORES COM LOCALIZAÇÃO
-----------------------------------

SELECT
    c.id,
    c.nome,
    c.marca,
    c.modelo,
    c.numero_serie,
    c.estado,
    l.nome AS localizacao_nome,
    l.descricao AS localizacao_descricao
FROM computadores c
LEFT JOIN localizacoes l
    ON c.localizacao_id = l.id;

Serve para listar PCs e mostrar onde estão.
O LEFT JOIN permite que apareçam também PCs sem localização.

9. VER COMPUTADORES DE UM INVENTÁRIO COM LOCALIZAÇÃO E RESPONSÁVEL
-------------------------------------------------------------------

SELECT
    c.id AS computador_id,
    c.nome AS computador_nome,
    c.marca,
    c.modelo,
    c.numero_serie,
    c.estado,

    l.id AS localizacao_id,
    l.nome AS localizacao_nome,
    l.descricao AS localizacao_descricao,

    u.id AS responsavel_id,
    u.nome AS responsavel_nome,
    u.email AS responsavel_email
FROM computadores c
LEFT JOIN localizacoes l
    ON c.localizacao_id = l.id
LEFT JOIN utilizadores u
    ON c.utilizador_responsavel_id = u.id
WHERE c.inventario_id = 1;

Esta é uma das queries mais úteis.
Ela devolve literalmente os PCs do inventário com localização e responsável.

39. COMPARAR COMPUTADORES REGISTADOS COM DISPOSITIVOS DESCOBERTOS
------------------------------------------------------------------

SELECT
    c.id AS computador_id,
    c.nome AS computador_nome,
    c.numero_serie,
    d.id AS dispositivo_id,
    d.ip,
    d.hostname,
    d.mac_address,
    CASE
        WHEN d.id IS NULL THEN 'Não encontrado no scan'
        ELSE 'Encontrado no scan'
    END AS estado_scan
FROM computadores c
LEFT JOIN dispositivos_descobertos d
    ON c.nome = d.hostname
   AND c.inventario_id = d.inventario_id
WHERE c.inventario_id = 1;

Esta é uma das melhores queries para um projeto de redes.
Mostra se cada PC registado apareceu ou não no scan.

49. VER PENDÊNCIAS DE UM INVENTÁRIO
-----------------------------------

SELECT
    c.id,
    c.nome,
    c.marca,
    c.modelo,
    c.numero_serie,
    c.estado,
    CASE
        WHEN c.localizacao_id IS NULL THEN 'Sem localização'
        WHEN c.utilizador_responsavel_id IS NULL THEN 'Sem responsável'
        WHEN c.nome IS NULL OR c.nome = '' THEN 'Sem nome'
        WHEN c.marca IS NULL OR c.marca = '' THEN 'Sem marca'
        WHEN c.modelo IS NULL OR c.modelo = '' THEN 'Sem modelo'
        WHEN c.numero_serie IS NULL OR c.numero_serie = '' THEN 'Sem número de série'
        WHEN c.estado IS NULL OR c.estado = '' THEN 'Sem estado'
        ELSE 'Sem pendência'
    END AS pendencia
FROM computadores c
WHERE c.inventario_id = 1
  AND (
        c.localizacao_id IS NULL
     OR c.utilizador_responsavel_id IS NULL
     OR c.nome IS NULL OR c.nome = ''
     OR c.marca IS NULL OR c.marca = ''
     OR c.modelo IS NULL OR c.modelo = ''
     OR c.numero_serie IS NULL OR c.numero_serie = ''
     OR c.estado IS NULL OR c.estado = ''
  );

Serve para listar PCs problemáticos dentro de um inventário.


50. VISÃO GERAL DO INVENTÁRIO - VERSÃO COM SUBQUERIES
-----------------------------------------------------

SELECT
    i.id AS inventario_id,
    i.nome AS inventario_nome,

    (
        SELECT COUNT(*)
        FROM computadores c
        WHERE c.inventario_id = i.id
    ) AS total_computadores,

    (
        SELECT COUNT(*)
        FROM computadores c
        WHERE c.inventario_id = i.id
          AND c.localizacao_id IS NULL
    ) AS computadores_sem_localizacao,

    (
        SELECT COUNT(*)
        FROM computadores c
        WHERE c.inventario_id = i.id
          AND c.utilizador_responsavel_id IS NULL
    ) AS computadores_sem_responsavel,

    (
        SELECT COUNT(*)
        FROM dispositivos_descobertos d
        WHERE d.inventario_id = i.id
    ) AS dispositivos_descobertos

FROM inventarios i
WHERE i.id = 1;

Esta query dá uma visão geral do inventário.
A versão com subqueries evita contagens repetidas causadas por vários LEFT JOINs.
